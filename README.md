#### 介绍
多级联动下拉菜单，也可单选可多选，支持中英文混合检索，
使用了jQuery.Hz2Py，支持拼音检索中文




#### 使用说明

必要文件：
jquery.js、
melist.css、
jQuery.Hz2Py-min.js、
melist.js


 **melist();//初始化** 
> 如果是json地址读取，class="soso"填写data-link="json地址"

 **linkageall(); //有json数据地址前提下直接执行**
